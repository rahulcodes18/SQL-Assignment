# SQL Project - Employee Department Queries

## 📌 Project Name
SQL Queries Practice (Employee/Department Database)

---

## 📂 About Project
This project contains basic SQL queries for practicing:
- Data retrieval
- Sorting
- Aggregation
- Joins (Self Join)
- Duplicate handling

---

## 🛠️ Database Used
- Database Name: company_db (or department_db)
- Table: Department / Employee (as per assignment)

---

## 📊 Queries Covered

1. Top 5 Highest Salary Records  
2. Department Wise Employee Count  
3. Second Highest Salary  
4. Salary Greater Than Average Salary  
5. Inner Join (Self Join)  
6. Left Join (Self Join)  
7. Group By with Having Clause  
8. Last 6 Months Records  
9. Find Duplicate Records  
10. Delete Duplicate Records
